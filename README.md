#  채용 서비스 백엔드 서버 리팩토링

## 목표
3-Layered Architecture와 테스트 코드를 적용하여 유지보수성과 신뢰성을 높입니다.

## 개발환경
1. **웹 프레임워크**: Express.js
2. **패키지 매니저**: yarn
3. **모듈 시스템**: ESM(ES6 모듈 시스템)
4. **데이터베이스**: MySQL
5. **ORM**: Prisma
